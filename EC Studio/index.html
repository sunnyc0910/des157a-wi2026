EC Studio Process Visualizations

March 17, 2026

With these design comps, my main goal was to move away from a generic space pixel style and create a retro-future "Space Journey" theme that feels like a complete game visual design. I developed a custom color palette using high-contrast pastels and vibrant neons to give the project an energetic, playful vibe.

I also created a series of original character illustrations that serve as the project's visual anchors. By refining the layout, the “final layout” is more of a combination of the first three versions. The background color matches the submission box’s background, which helps balance the user's focus on the illustration.
